## Overview

Aeronautics Create SMP is a multiplayer-focused modpack built around the **Create** ecosystem and **Create Aeronautics**, tuned for players who want factories, transport networks, and flying contraptions to matter in long-running survival worlds.

The pack is structured around a simple idea: push engineering gameplay further without turning progression into noise. That means automation, logistics, aviation, mapping, and server-friendly configuration choices all work together as a single technical stack instead of a random pile of mods.

## Stack Snapshot

- **Platform:** Minecraft `1.21.1`
- **Mod Loader:** NeoForge `21.1.225`
- **Package Format:** Modrinth `.mrpack`
- **Release:** `1.3.0`
- **Content Volume:** `218` mods, `13` resource packs, `22` shader packs

## Design Goals

- Build large-scale mechanical systems around the Create ecosystem.
- Support airship-driven exploration and transport as first-class gameplay.
- Keep the pack stable enough for persistent SMP hosting.
- Extend progression with curated addons instead of uncontrolled bloat.
- Preserve performance and compatibility for dedicated server deployment.

## Included Systems

- **Automation layer:** assembly lines, rotational power, logistics, processing chains
- **Aerial layer:** Create Aeronautics-style flying contraptions and mobile infrastructure
- **World support:** mapping, navigation, exploration helpers, survival utility mods
- **Server layer:** packaged overrides and config tuning intended for multiplayer operation

## Quick Start

1. Install a launcher with Modrinth pack support.
2. Import [`Aeronautics 1.3.0.mrpack`](./Aeronautics%201.3.0.mrpack).
3. Launch with NeoForge for Minecraft `1.21.1`.
4. Use the included overrides and configs as the base for your SMP deployment.

## Why This Pack

Most Create packs stop at "more addons." Aeronautics Create SMP is aimed at a more technical experience: machines should scale, transport should be engineered, and collaborative infrastructure should feel like part of the world architecture rather than a side activity.

If your ideal server ends up with dockyards, freight lines, processing plants, flight routes, and player-built industrial districts, this pack is pointed at exactly that.
